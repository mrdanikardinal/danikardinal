Belajar CSS

Selamat datang di dokumentasi interaktif CSS.

CSS adalah bahasa yang kita gunakan untuk menata halaman Web.

Sejarah CSS

CSS, atau Cascading Style Sheets, telah berevolusi melalui beberapa generasi untuk meningkatkan kemampuan dalam mengatur tampilan halaman web. Awalnya, CSS diusulkan oleh Håkon Wium Lie pada tahun 1994 di CERN, tempat lahirnya web, saat bekerja dengan Tim Berners-Lee. CSS kemudian distandarisasi oleh W3C, dengan CSS1 dirilis pada tahun 1996, diikuti oleh CSS2 pada tahun 1998. CSS3, yang dirilis secara modular, membawa fitur-fitur baru dan lebih canggih. Saat ini, CSS terus berkembang dengan standar seperti CSS4 yang sedang dalam tahap pengembangan.

Berikut adalah garis waktu evolusi CSS:

Pra-CSS:

Sebelum CSS, pengaturan tampilan halaman web dilakukan dengan tag-tag HTML, yang menyebabkan kode menjadi rumit dan sulit dikelola.

CSS1 (1996):

CSS1 memungkinkan pemisahan konten (HTML) dan presentasi (CSS), membuat kode lebih terstruktur dan mudah dikelola.

CSS2 (1998):

CSS2 memperluas kemampuan CSS, termasuk penataan tata letak yang lebih kompleks dan dukungan untuk media yang berbeda (seperti printer).

CSS3 (Modular):

CSS3 memperkenalkan konsep modularitas, memungkinkan pengembangan fitur-fitur baru secara terpisah dan lebih cepat.

CSS4 (Dalam Pengembangan):

CSS4 menghadirkan fitur-fitur yang lebih canggih seperti variabel CSS, CSS Grid Layout, dan dukungan dark mode.
