Belajar CSS

Apa Itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan gaya elemen dalam dokumen HTML, seperti warna, ukuran, margin, dan layout.

Manfaat Menggunakan CSS

CSS memisahkan konten dari tampilan, sehingga memudahkan pengelolaan dan konsistensi desain antar halaman website.

Jenis-Jenis CSS

Ada tiga jenis CSS: inline (langsung di elemen), internal (dalam tag <style> di HTML), dan eksternal (file .css terpisah).

Struktur Penulisan CSS

Struktur dasar CSS terdiri dari selector, property, dan value. Contoh: p { color: blue; } akan mewarnai teks paragraf menjadi biru.

Selector di CSS

Selector digunakan untuk memilih elemen HTML yang akan diberi gaya, misalnya menggunakan nama tag, class, ID, atau kombinasi.

Properti Umum di CSS

Beberapa properti CSS yang sering digunakan meliputi color, font-size, margin, padding, dan background.

Class dan ID dalam CSS

Class digunakan untuk memberi gaya pada banyak elemen, sedangkan ID ditujukan untuk satu elemen unik. Penulisan class diawali titik (.) dan ID diawali tanda pagar (#).

Responsive Design dengan CSS

CSS memungkinkan website menjadi responsif menggunakan media queries, sehingga tampilan dapat menyesuaikan berbagai ukuran layar.

Penggunaan Framework CSS

Framework seperti Bootstrap dan Tailwind CSS membantu mempercepat proses styling dengan menyediakan kelas-kelas siap pakai.

Tips Belajar CSS

Mulailah dari konsep dasar, praktik langsung membuat halaman web, dan eksplorasi dokumentasi seperti MDN atau W3Schools.
